Web Layout

Tranditional flow of HTML elements are to be be stacked on top of each other. These Types of elements are known as block elements and they start on a new line while taking up the full width available.

Inline elements are those that do not start on a new line and only take up as much width as necessary. A good example of this is the span tag. This tag will not take up specified width or height values.

Inline-block elements are those that flow flow with text and do NOT start on a new line. They also maintian their size and shape like block elements.

Float can be used to move elements to the left or right of the page and allow text to wrap around it.
